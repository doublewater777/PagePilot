# Engineering Plan: PagePilot Pro Implementation

Generated by /plan-eng-review on 2026-05-30
Branch: main
Repo: yangpan/PagePilot
Design: PagePilot Pro — 统计即 Pro

## Architecture Decisions (Locked)

1. **Pro 状态持久化：** UserDefaults 缓存 (`entitlements_isPro`) + 启动时 StoreKit 2 `currentEntitlements` 异步验证
2. **StoreKit 管理器：** `ProPurchaseManager` 纯 Swift 单例 + `NotificationCenter` (`proAccessDidChange`)
3. **Grandfathering：** 不设，所有用户从 1.1 起统一看到 Paywall
4. **iPad 支持：** 第一版不做 iPad 锁定，聚焦"统计即 Pro"
5. **Paywall 触发：** 本地 Paywall 页 (`PaywallView`) 展示价值后再购买
6. **Pro offer model：** 月度、年度、终身三档；年度订阅为默认推荐，终身为一次性购买选项。

### 1. `Sources/Store/ProPurchaseManager.swift`
StoreKit 2 购买管理单例。

```swift
import StoreKit
import Foundation

enum ProPurchaseError: Error {
    case productNotFound
    case purchaseFailed
    case verificationFailed
}

final class ProPurchaseManager {
    static let shared = ProPurchaseManager()
    
    static let proAccessDidChange = Notification.Name("proAccessDidChange")
    
    static let monthlyProductID = "com.panyang.PagePilot.pro.monthly"
    static let yearlyProductID = "com.panyang.PagePilot.pro.yearly"
    static let lifetimeProductID = "com.panyang.PagePilot.pro.lifetime"
    private let defaults = UserDefaults.standard
    private let proKey = "entitlements_isPro"
    
    private var updateListenerTask: Task<Void, Never>?
    private(set) var products: [Product] = []
    
    var hasProAccess: Bool {
        defaults.bool(forKey: proKey)
    }
    
    private init() {
        updateListenerTask = listenForTransactions()
        Task { await loadProducts() }
    }
    
    deinit {
        updateListenerTask?.cancel()
    }
    
    // MARK: - Product Loading
    
    func loadProducts() async {
        do {
            products = try await Product.products(for: [
                Self.monthlyProductID,
                Self.yearlyProductID,
                Self.lifetimeProductID,
            ])
        } catch {
            print("Failed to load products: \(error)")
        }
    }
    
    // MARK: - Purchase
    
    func purchase(_ product: Product) async throws {
        let result = try await product.purchase()
        
        switch result {
        case .success(let verification):
            let transaction = try checkVerified(verification)
            await transaction.finish()
            await syncCurrentEntitlements()
            
        case .userCancelled:
            throw ProPurchaseError.purchaseFailed
            
        case .pending:
            throw ProPurchaseError.purchaseFailed
            
        @unknown default:
            throw ProPurchaseError.purchaseFailed
        }
    }
    
    // MARK: - Restore
    
    func restorePurchases() async {
        do {
            try await AppStore.sync()
            await syncCurrentEntitlements()
        } catch {
            print("Restore failed: \(error)")
        }
    }
    
    // MARK: - Verification
    
    func verifyCurrentEntitlements() async {
        await syncCurrentEntitlements()
    }
    
    // MARK: - Helpers
    
    private func listenForTransactions() -> Task<Void, Never> {
        Task.detached(priority: .background) { [weak self] in
            for await result in Transaction.updates {
                guard let self = self else { return }
                do {
                    let transaction = try self.checkVerified(result)
                    if Self.proProductIDs.contains(transaction.productID) {
                        await self.syncCurrentEntitlements()
                    }
                    await transaction.finish()
                } catch {
                    print("Transaction update failed: \(error)")
                }
            }
        }
    }
    
    private func checkVerified<T>(_ result: VerificationResult<T>) throws -> T {
        switch result {
        case .unverified(_, let error):
            throw error
        case .verified(let safe):
            return safe
        }
    }
    
    @discardableResult
    private func syncCurrentEntitlements() async -> Bool {
        var hasPro = false
        for await result in Transaction.currentEntitlements {
            guard let transaction = try? checkVerified(result),
                  Self.proProductIDs.contains(transaction.productID)
            else { continue }
            hasPro = true
            break
        }

        await updateProAccess(hasPro)
        return hasPro
    }
    
    @MainActor
    private func updateProAccess(_ hasAccess: Bool) async {
        defaults.set(hasAccess, forKey: proKey)
        NotificationCenter.default.post(name: Self.proAccessDidChange, object: nil)
    }
}

extension ProPurchaseManager {
    static var proProductIDs: Set<String> {
        [monthlyProductID, yearlyProductID, lifetimeProductID]
    }
}
```

### 2. `Sources/Home/PaywallView.swift`
本地 Paywall 页，展示 Pro 价值后触发购买。

**设计决策（from /plan-design-review）：**
- Crown icon 保持（与 Apple Watch Digital Crown 有天然关联）
- Feature list 用 `cardStyle()` 包裹（与 HomeView 风格一致）
- CTA button 用 `AppColors.accentGradient`（蓝到紫，与现有设计系统一致）
- 内容最大宽度 420（iPad 上居中，不拉伸）
- 购买成功后显示 "🎉 已解锁" 提示再 dismiss

```swift
import SwiftUI
import StoreKit

struct PaywallView: View {
    @Environment(\.dismiss) private var dismiss
    @State private var isPurchasing = false
    @State private var purchaseError: Error?
    @State private var showError = false
    @State private var showSuccess = false
    
    var body: some View {
        NavigationView {
            ScrollView {
                VStack(spacing: 28) {
                    // Hero
                    VStack(spacing: 12) {
                        Image(systemName: "crown.fill")
                            .font(.system(size: 48))
                            .foregroundColor(.yellow)
                        
                        Text(NSLocalizedString("paywall_title", comment: ""))
                            .font(.system(size: 24, weight: .bold))
                        
                        Text(NSLocalizedString("paywall_subtitle", comment: ""))
                            .font(.system(size: 15))
                            .foregroundColor(.secondary)
                            .multilineTextAlignment(.center)
                    }
                    .padding(.top, 20)
                    
                    // Features (wrapped in card style)
                    VStack(alignment: .leading, spacing: 16) {
                        featureRow(icon: "chart.bar.fill", text: NSLocalizedString("paywall_feature_stats", comment: ""))
                        featureRow(icon: "flame.fill", text: NSLocalizedString("paywall_feature_streak", comment: ""))
                        featureRow(icon: "arrow.up.arrow.down", text: NSLocalizedString("paywall_feature_export", comment: ""))
                    }
                    .padding(20)
                    .background(Color(.systemBackground))
                    .cornerRadius(20)
                    .overlay(
                        RoundedRectangle(cornerRadius: 20)
                            .stroke(Color.primary.opacity(0.1), lineWidth: 1)
                    )
                    .shadow(color: Color.black.opacity(0.04), radius: 16, x: 0, y: 6)
                    .padding(.horizontal, 24)
                    
                    // Pricing
                    VStack(spacing: 8) {
                        if let product = ProPurchaseManager.shared.product {
                            Text(product.displayPrice)
                                .font(.system(size: 32, weight: .bold))
                            Text(NSLocalizedString("paywall_one_time", comment: ""))
                                .font(.system(size: 13))
                                .foregroundColor(.secondary)
                        } else {
                            Text(NSLocalizedString("paywall_loading_price", comment: ""))
                                .font(.system(size: 13))
                                .foregroundColor(.secondary)
                        }
                    }
                    .padding(.top, 8)
                    
                    // CTA with accent gradient
                    Button(action: purchase) {
                        HStack {
                            if isPurchasing {
                                ProgressView()
                                    .progressViewStyle(CircularProgressViewStyle(tint: .white))
                            } else {
                                Text(NSLocalizedString("paywall_buy_button", comment: ""))
                            }
                        }
                        .frame(maxWidth: .infinity)
                        .padding(.vertical, 18)
                        .background(
                            LinearGradient(
                                colors: [
                                    Color(red: 0.38, green: 0.56, blue: 1.0),
                                    Color(red: 0.56, green: 0.44, blue: 0.96)
                                ],
                                startPoint: .leading,
                                endPoint: .trailing
                            )
                        )
                        .foregroundColor(.white)
                        .cornerRadius(14)
                    }
                    .disabled(isPurchasing)
                    .padding(.horizontal, 24)
                    
                    // Restore
                    Button(NSLocalizedString("paywall_restore_button", comment: "")) {
                        Task { await restore() }
                    }
                    .font(.system(size: 14))
                    .foregroundColor(.blue)
                    
                    Spacer()
                }
                .frame(maxWidth: 420)
                .frame(maxWidth: .infinity)
            }
            .navigationBarTitleDisplayMode(.inline)
            .toolbar {
                ToolbarItem(placement: .navigationBarTrailing) {
                    Button(NSLocalizedString("paywall_close", comment: "")) {
                        dismiss()
                    }
                }
            }
            .alert(NSLocalizedString("paywall_error_title", comment: ""), isPresented: $showError) {
                Button(NSLocalizedString("ok_button", comment: ""), role: .cancel) {}
            } message: {
                Text(purchaseError?.localizedDescription ?? "")
            }
            .overlay(
                Group {
                    if showSuccess {
                        VStack(spacing: 12) {
                            Image(systemName: "checkmark.circle.fill")
                                .font(.system(size: 48))
                                .foregroundColor(.green)
                            Text(NSLocalizedString("paywall_success_title", comment: ""))
                                .font(.system(size: 18, weight: .bold))
                        }
                        .padding(24)
                        .background(Color(.systemBackground).opacity(0.95))
                        .cornerRadius(16)
                        .shadow(radius: 20)
                    }
                }
            )
        }
    }
    
    private func featureRow(icon: String, text: String) -> some View {
        HStack(spacing: 12) {
            Image(systemName: icon)
                .font(.system(size: 20))
                .foregroundColor(
                    Color(red: 0.38, green: 0.56, blue: 1.0)
                )
                .frame(width: 28)
            Text(text)
                .font(.system(size: 15))
            Spacer()
        }
    }
    
    private func purchase() {
        isPurchasing = true
        Task {
            do {
                try await ProPurchaseManager.shared.purchase()
                withAnimation {
                    showSuccess = true
                }
                try await Task.sleep(nanoseconds: 1_500_000_000)
                dismiss()
            } catch {
                purchaseError = error
                showError = true
            }
            isPurchasing = false
        }
    }
    
    private func restore() async {
        await ProPurchaseManager.shared.restorePurchases()
        if ProPurchaseManager.shared.hasProAccess {
            withAnimation {
                showSuccess = true
            }
            try? await Task.sleep(nanoseconds: 1_500_000_000)
            dismiss()
        }
    }
}
```

## Test Plan

### Unit Tests
1. `ProPurchaseManagerTests`:
   - `hasProAccess` 返回 UserDefaults 值
   - `syncCurrentEntitlements()` 根据 verified current entitlements 同步 UserDefaults
   - entitlement 为空、过期或撤销后 `updateProAccess(false)` 清除本地 Pro 状态

### Integration Tests (StoreKit Test Configuration)
1. 购买成功流程：点击 Paywall → purchase() → transaction verified → UserDefaults 写入 → UI 刷新
2. 购买取消流程：点击 Paywall → userCancelled → 不写入 UserDefaults
3. 恢复购买流程：点击 Restore → currentEntitlements 有记录 → 写入 UserDefaults
4. 启动验证流程：App 启动 → verifyCurrentEntitlements → 同步 UserDefaults 和 Transaction 状态

### UI Tests
1. HomeView 非 Pro 用户：切换统计 scope 到 Week → 显示 locked view → 点击升级 → 弹出 Paywall
2. HomeView Pro 用户：切换统计 scope 到 Week → 直接显示统计内容
3. SettingsView 非 Pro 用户：显示"升级 Pro"行 → 点击 → 弹出 Paywall
4. SettingsView Pro 用户：显示"Pro 已解锁"

## Failure Modes

| 场景 | 测试覆盖 | 错误处理 | 用户体验 |
|------|----------|----------|----------|
| StoreKit product 加载失败 | 无（需要网络）| 打印日志 | Paywall 显示"加载中" |
| 购买过程中网络断开 | 有（StoreKit Test）| catch error | 显示错误 alert |
| 用户取消购买 | 有（StoreKit Test）| 不抛错，静默 | 留在 Paywall |
| 购买待处理（家长控制）| 有（StoreKit Test）| 标记 pending | 提示"等待家长批准" |
| Transaction 验证失败 | 有 | 抛错 | 显示错误 alert |
| 重装 App 后恢复购买 | 有 | verifyCurrentEntitlements | 启动时静默恢复 |
